# Gunslinger zig example
 
 Example project using the [zig](https://ziglang.org/) programming language with the [gunslinger](https://github.com/MrFrenik/gunslinger) framework.
 
 compile and run with: zig build run
 
 Example includes:
 - Immediate draw util
 - Pipeline
 - Uniform
 - GUI
